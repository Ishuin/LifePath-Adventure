## 2026-06-06 - Keyboard Accessibility & ARIA\n**Learning:** This Vite/React application lacked clear focus indicators for interactive elements and had decorative canvas animations without ARIA attributes, hindering keyboard navigation and screen reader compatibility.\n**Action:** Implemented global `:focus-visible` styles for navigation links and buttons, added `aria-hidden="true"` to the decorative canvas wrapper, and added smooth scrolling behavior to enhance overall accessibility and user experience.
